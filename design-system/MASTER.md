## UI STYLE GUIDE

### Dashboard Layout (Data-Dense)
```css
display: grid;
grid-template-columns: repeat(12, 1fr);
gap: 8px;
padding: 12px;
font-size: 12-14px;        /* compact readable */
--sidebar-width: 240px;
--header-height: 56px;
--card-padding: 12px;
--table-row-height: 36px;
```

**Dashboard checklist:**
- [ ] 12-column grid
- [ ] KPI cards top row (glucose level, trend, last reading, streak)
- [ ] Tables sortable with sticky headers
- [ ] Real-time chart with pulse animation
- [ ] Alert toast system (critical/warning/normal states)
- [ ] Loading states for all data fetches
- [ ] Export functionality

### Landing Page (Scroll-Triggered Storytelling)
```
1. Intro hook          — "What if managing glucose felt effortless?"
2. Chapter 1 (Problem) — Daily friction, glucose monitoring pain points
3. Chapter 2 (Journey) — How it works, your data, your story
4. Chapter 3 (Solution)— The app/platform features
5. Climax CTA          — "Start your free trial" / "Book a demo"
```

**GSAP ScrollTrigger per chapter:** Each chapter pins, scrubs into focus, then releases.

### Component Standards
- Cards: `1px border` using `--border`, `0px shadow` → `4px shadow` on hover (spring)
- Buttons: Primary = `--primary` bg, `--primary-foreground` text, `border-radius: 8px`
- Inputs: `--muted` bg, `--border` border, focus ring uses `--ring`
- Tables: `--muted` alternate rows, sticky `--card` headers, sortable columns
- Alerts: position fixed, z-index 1000, pulse animation for live status

## ANIMATION SYSTEM (GSAP)

Install: `npm install gsap` (ALL plugins free — no auth needed)

### Core patterns for this project
```js
// Glucose chart real-time update
gsap.to(chartLine, { 
  drawSVG: "0% 100%", 
  duration: 0.8, 
  ease: "power2.out" 
})

// Alert pulse (critical reading)
gsap.to(alertDot, {
  scale: 1.3, opacity: 0.6,
  repeat: -1, yoyo: true,
  duration: 0.8, ease: "power1.inOut"
})

// ScrollTrigger landing chapter
gsap.fromTo(".chapter-2", 
  { opacity: 0, y: 60 },
  { opacity: 1, y: 0, duration: 1, ease: "power3.out",
    scrollTrigger: { trigger: ".chapter-2", start: "top 80%" }
  }
)
```

## ANTI-PATTERNS FOR THIS PROJECT

- No neon colors — medical trust requires calm palette
- No centered hero — `DESIGN_VARIANCE: 8` enforced
- No autoplay video — `playsInline muted preload='none'` only
- No Inter as primary font — use Figtree/Plus Jakarta Sans
- No fake loading — show real loading states for glucose data fetches
- Bulk actions required for data tables — no single-row-only actions
